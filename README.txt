Type make to create the library libgrid.a as well as the three executable.

You may type ./test to run automated tests with their own check.

You may type ./play with the parameter terminal or ncurses to use the said interface, an usage function prevents you from playing it without parameter.
An other executable can be launched with ./2048, it let you play the game in graphic mode.
During the game commands are given to you.
To play again you must re-enter the command.

Type make clean to return to the original state.
